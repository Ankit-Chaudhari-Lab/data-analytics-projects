# Automobile Data Analysis Dashboard

## Project Overview
This project analyses automobile data to uncover insights on car pricing, performance, and specifications. Using Python for data cleaning and exploratory analysis, and Power BI for interactive dashboards, it helps visualize key metrics such as car prices by brand, power-to-weight ratio, body styles, and car types. The insights can support pricing strategies, market research, and consumer decision-making.

## Tools Used
- **Power BI** – Interactive dashboards, KPI visualization, filtering, and trend analysis  
- **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning, transformation, and EDA  
- **Excel / SQL** – Data preparation and querying  

## Key Insights
- Price trends across different car brands  
- Power-to-weight ratio and performance analysis  
- Distribution of car types and body styles  
- Identification of key factors affecting car pricing
